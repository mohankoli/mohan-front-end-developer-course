# Vector Similarity Search (Dot Product Ranking)

## Problem
Given a list of documents with vector embeddings and a query vector, find the most similar documents.

Similarity is calculated using the **dot product**.

---

## Example

Documents

documents = [
 {id:1, vector:[0.2,0.8,0.5]},
 {id:2, vector:[0.9,0.1,0.2]},
 {id:3, vector:[0.3,0.7,0.6]}
]

Query

query = [0.25,0.75,0.55]

k = 3

---

## Approach

1. Iterate through all documents
2. Compute similarity score using **dot product**
3. Score formula:

score = Σ(query[i] × documentVector[i])

4. Store `{id, score}`
5. Sort results by score (descending)
6. Return top K documents

---

## JavaScript Implementation

```javascript
let documents = [
 {id:1, vector:[0.2,0.8,0.5]},
 {id:2, vector:[0.9,0.1,0.2]},
 {id:3, vector:[0.3,0.7,0.6]}
]

let query = [0.25,0.75,0.55]
let k = 3

function search(documents, query, k) {

   let res = documents.map((doc)=>{

       let score = 0;

       for(let i=0;i<query.length;i++){
           score += query[i] * doc.vector[i];
       }

       return {id:doc.id, score}
   })
   .sort((a,b)=> b.score - a.score)

   return res.slice(0,k)
}

console.log(search(documents, query, k))
