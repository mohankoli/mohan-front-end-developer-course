# LRU Cache (Least Recently Used)

## Problem
Implement a simple **LRU Cache** with a fixed capacity.

When the cache reaches its capacity:
- The **least recently used item** should be removed.

---

## Example

Capacity = 3

Operations

set(1)  
set(2)  
set(3)  
set(4)

Result

[4,3,2]

Explanation:

1 → [1]  
2 → [2,1]  
3 → [3,2,1]  
4 → [4,3,2] (1 removed because it is least recently used)

---

## Approach

1. Maintain a list (`store`) to keep cache items
2. When inserting a new item:
   - If item already exists → remove it
   - If capacity reached → remove last element
3. Insert new item at the beginning (most recently used)

---

## JavaScript Implementation

```javascript
class LRU {
    constructor(capacity){
        this.capacity = capacity;
        this.store = [];
    }
    
    set(item) {
        let index = this.store.indexOf(item);
        
        if(index !== -1) {
            this.store.splice(index,1);
        } 
        else if(this.store.length === this.capacity) {
            this.store.pop();
        }
        
        this.store.unshift(item);
        return this.store;
    }
}

let lr = new LRU(3);

console.log(lr.set(1));
console.log(lr.set(2));
console.log(lr.set(3));
console.log(lr.set(4));
