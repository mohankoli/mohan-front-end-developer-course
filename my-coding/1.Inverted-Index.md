# Inverted Index

## Problem
Build an inverted index from a list of documents.

An inverted index maps:

word → list of document IDs

Example

docs = [
 {id:1, text:"search engine"},
 {id:2, text:"search system"},
 {id:3, text:"document retrieval"}
]

Output

{
 search: [1,2],
 engine: [1],
 system: [2],
 document: [3],
 retrieval: [3]
}

---

# Approach

1. Iterate through all documents
2. Split document text into words
3. For each word
   - store document ID in index
4. Return index

---

# JavaScript Implementation

```javascript
function buildIndex(docs){

    let index = {};

    for(let {id,text} of docs){

        let words = text.split(" ");

        for(let word of words){

            if(!index[word]){
                index[word] = [];
            }

            index[word].push(id);
        }
    }

    return index;
}
